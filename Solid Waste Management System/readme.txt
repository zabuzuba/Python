To run the project:

1) Unzip the files
2) Download the model from the below link
https://drive.google.com/file/d/1RfxC9ARIzjH20W4kSyExVfYTg46IyXfs/view?usp=share_link
3) Make sure to keep the model and the application files in the same folder
4) Run the app.py
5) It will prompt a web link
6) Copy the link and search it in any web browser 
7) Now you can upload any image of your choice to classify it.
